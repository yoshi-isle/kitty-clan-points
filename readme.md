Local development

Welcome to the kitty clan points bot project

1. Navigate to clan-points-bot
2. Copy 'example.env' and create '.env'
3. Fill in the values (lmao)
4. Run docker compose watch at the root. This will start the bot and create a local Mongo database. This doesnt not create the collections (yet)
5. Get a sheets_config.json from google cloud API for google sheets (lmao)
